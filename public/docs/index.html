Informações Gerais

  • Base URL: /api/
  • Formato de Resposta: JSON
  • Métodos HTTP: GET (somente leitura)
  • Autenticação: Não requerida para endpoints públicos
  • Paginação: Automática com limite configurável
  • Filtros: Parâmetros query para busca e filtros
  • Eager Loading: Relacionamentos carregados automaticamente
  • Códigos de Status: 200 (sucesso), 404 (não encontrado), 500 (erro)
